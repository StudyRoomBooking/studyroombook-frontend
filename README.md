# Study Room Booking Frontend

Hello from branch 2

## 配置环境
使用nvm配置node
```
nvm use
```

下载packages
```
npm install
```

## Running

首先，运行开发server

```bash
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000) 访问网站

## 文件结构
```sh
pages/            # 
    _app.tsx
    about.tsx

src/
    app/
    components/    #组件
```
